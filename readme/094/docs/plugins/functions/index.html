Список функций, доступных для плагинов

В плагинах вы можете использовать следующие функции:
Название Краткое описание
generatePageLink() Генерация ссылки на конкретную страницу в постраничной навигации
generatePagination() Генерация постраничной навигации
generatePluginLink() Генерация ссылки на заданный обработчик ссылок заданного плагина
